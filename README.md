# AppleTree
Apple Tree is an App for MacOS & Windows (hopefully Linux) that creates a visual directory (tree) structure for any application of your choosing, allowing you to evaluate applications file structures in depth to compare with other trees.

### How it works
Apple Tree lets you select apps from the application folder or by dragging and dropping an app into the drop box. It then lets you creates a .txt file with the tree structure, its that simple!

### Example
```
Apple Tree
└── Contents
    ├── Apples
    ├── Info.plist
    ├── MacOS
    │   ├── Apple Tree
    │   ├── Apple Tree.debug.dylib
    │   └── __preview.dylib
    ├── PkgInfo
    ├── Resources
    │   ├── 128.png
    │   ├── 16.png
    │   ├── 256.png
    │   ├── 32.png
    │   ├── 512.png
    │   ├── 64.png
    │   ├── App Icon.png
    │   ├── AppIcon.icns
    │   └── Assets.car
    └── _CodeSignature
        └── CodeResources

```
